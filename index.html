YouTube API Key 설정

이 웹앱은 사용자의 브라우저에서 직접 데이터를 가져옵니다. 본인의 API 키를 입력해주세요.

API 키 발급 방법을 모르시나요? (가이드 열기)
  1. Google Cloud Console에 접속 및 구글 계정으로 로그인합니다.
  2. 화면 최상단의 [프로젝트 선택]을 누른 뒤, 팝업창 우측 상단의 [새 프로젝트]를 클릭해 빈 프로젝트를 생성합니다.
  3. (중요) 방금 만든 새 프로젝트가 화면 상단에 제대로 선택(활성화)되어 있는지 한 번 더 확인합니다.
  4. 좌측 햄버거(三) 버튼을 눌러 [API 및 서비스] > [라이브러리] 메뉴로 찾아 들어갑니다.
  5. 검색창에 "YouTube Data API v3"를 검색해 클릭한 후, 파란색 [사용] 버튼을 누릅니다.
  6. 잠시 후 로딩이 끝나면 화면 좌측의 [사용자 인증 정보] 탭 메뉴로 이동합니다.
  7. 화면 상단의 [+ 사용자 인증 정보 만들기]를 누르고 [API 키]를 선택합니다.
  8. 생성된 영문/숫자 조합의 긴 API 키를 복사하여 아래 입력칸에 붙여넣습니다. (구글 API 키는 보통 "AIza..." 로 시작합니다)

※ 입력하신 키는 서버로 전송되지 않으며
브라우저(로컬 스토리지)에만 안전하게 보관됩니다.
